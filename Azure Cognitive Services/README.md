# Azure Cognitive Services Demo

Azure Cognitive Services are cloud-based services with REST APIs and client library SDKs available to help you build cognitive intelligence into your applications.

## Key Concepts for Interviews

- **Multi-Service Resource**: One key and endpoint for multiple services (Vision, Language, Speech).
- **Computer Vision**: Analyzing images (OCR, object detection).
- **Text Analytics**: Sentiment analysis, key phrase extraction.
- **Authentication**: Done via API Keys or Azure AD (Entra ID).

## Included Demos

1. **VisionSample.cs**: Example of analyzing an image using the Computer Vision SDK.
